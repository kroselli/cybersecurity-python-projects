# Security Log Analyzer

A Python script that analyzes a log file and detects suspicious authentication activity by identifying failed login attempts.

## Features
- Reads a text-based server log
- Detects failed login attempts
- Counts suspicious authentication events

## Files
- log_analyzer.py - main script
- server_log.txt - sample log file

## How to Run
python3 log_analyzer.py

## Concepts Demonstrated
- Security log analysis
- Authentication monitoring
- Basic threat detection
